# 🌐 Cyber-Physical Systems Learning Repository

Welcome to this comprehensive repository for learning **Cyber-Physical Systems (CPS)**, designed for progression from beginner to advanced levels across multiple CPS domains.

## 📚 Repository Structure

This repository is organized into specialized domains, each with its own learning path from **beginner → intermediate → advanced**:

### **Core CPS Domains**

| Domain | Focus | Primary Languages | Level |
|--------|-------|-------------------|-------|
| 🤖 [Robotics](robotics/) | Robot design, control, ROS | C++, Python | All |
| 🚗 [Autonomous Vehicles](autonomous-vehicles/) | Self-driving cars, perception, control | C++, Rust, Python | All |
| 🚗 [Voiture Autonome](voiture_autonome/) | Autonomous vehicles (French) | C++, Python | All |
| 📡 [Sensors](sensors/) | Sensor technology, signal processing | Python, C++ | All |
| 🌐 [IoT](iot/) | Internet of Things, connectivity | Python, C++, JS | All |
| ⚡ [Edge Computing](edge-computing/) | Edge devices, Kubernetes, ML at edge | C++, Rust, Python | All |
| 🔒 [Network Security](network-security/) | Cybersecurity, encryption, threat detection | Rust, C++, Python | All |
| ⏱️ [Real-Time Systems](real-time-systems/) | Scheduling, RTOS, timing constraints | C++, Rust | All |
| 🔗 [Connectivity Protocols](connectivity-protocols/) | CAN, Modbus, MQTT, 5G | C++, Python | All |
| 🏭 [Industrial Control](industrial-control/) | PLCs, SCADA, Industry 4.0 | C++, Python | All |
| ⚡ [Smart Grids](smart-grids/) | Power systems, renewable energy | Python, C++ | All |
| 🏙️ [Smart Cities](smart-cities/) | Urban IoT, AI services | Python, C++, JS | All |
| 🏥 [Medical Devices](medical-devices/) | Healthcare CPS, regulatory compliance | C++, Rust, Python | All |

## 🎯 How to Use This Repository

### **For Beginners**
1. Start with any domain that interests you
2. Read the **README.md** in that folder for learning path
3. Follow the **BEGINNER LEVEL** section
4. Work through the projects in order
5. Gradually progress to intermediate level

### **For Intermediate Learners**
1. Choose a domain to specialize in
2. Review the **INTERMEDIATE LEVEL** concepts
3. Build more complex projects
4. Integrate with other domains
5. Start contributing to open-source

### **For Advanced Learners**
1. Explore **ADVANCED LEVEL** projects
2. Implement cutting-edge technologies (AI/ML, blockchain, 5G)
3. Work on safety-critical systems
4. Focus on specific certifications (ISO 26262, IEC 60601, etc.)
5. Contribute domain expertise

## 🚀 Getting Started

### **Prerequisites**
- Linux (Ubuntu 20.04 or later)
- Git
- Docker
- Python 3.8+
- Basic programming knowledge

### **Quick Start**
```bash
# Clone the repository
git clone https://github.com/MELAI-1/cyberphysical-systems.git
cd cyberphysical-systems

# Choose a domain and start learning
cd robotics
cat README.md

# Set up your development environment
docker-compose up -d
```
